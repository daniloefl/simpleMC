int Process::getNdof ( )
virtual

Returns the number of degrees of freedom for a scattering.

Number of output degrees of freedom = 4*N - N -4 For N final state particles 4*N = each particle has 4 components in the four-vector -N = they are all on-shell (their mass is known) -4 = the system conserves energy-momentum (4 conservation equations) This should be overwritten depending if the processes is 2->2, 2->3, etc

Returns
ndof

void Process::getOut2to2 ( FourVector  vb1,
FourVector  vb2,
std::vector< double > &  x,
std::vector< FourVector > &  o 
)
virtual

Generates output vectors in case of a e+ e- -> e+ e- with uniform random theta_1 and phi_1 between 0 and pi.

One can estimate the final state with only 2 angles in a 2->2 process Energy conservation provides 4 relations from the initial state to the final state vectors.

This needs to be generalised for any other process (not only ee -> ee)

Parameters
b1First beam vector
b2Second beam vector
xVector with two elements that are random numbers between 0 and 1: this uniquely specifies the final state
oVector of output 4-vectors to be returned by reference.

The particles are assumed to be on-shell, so the mass is known, which imposes a constrain in the energy of each final state particle. Those count 6 equations on 8 unknowns, so only 2 angles are missing. The vector x is this vector of random unknowns, which will be the start of the MC iteration procedure.
